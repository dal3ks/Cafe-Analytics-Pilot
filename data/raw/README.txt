# Raw Data Folder

**Notice:**  
This folder is intentionally empty. It is a placeholder for raw input data files that **contain personally identifiable information (PII)** or other sensitive data.

To maintain privacy and GDPR compliance, **no raw or unprocessed data files are included in this repository**.

---

### How to Use Locally

If you are running the project locally:
1. Place the provided `raw_data.txt` file in this folder.
2. Run the `cleaning_script.py` located in `src/` to generate an anonymised dataset (`cleaned_data.csv`) in `data/processed/`.
