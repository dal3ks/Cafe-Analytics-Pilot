# Documentation Folder

This folder contains all supporting materials for the **Cafe Analytics Pilot** project.

---

### Contents

- **architecture_diagram.png** – Visual representation of the project workflow (data flow from raw input → Python cleaning → Power BI visualisation).  
- **whiteboard_session.png** – Snapshot of project planning and brainstorming.  
- **Cafe_Analytics_Pilot_Presentation.pptx** – Final project presentation slides.  

---

### Purpose

This folder centralises all **non-code documentation**:
- Design notes and planning materials  
- Project architecture diagrams  
- Presentation decks or visual summaries  
- Any additional written documentation or reports

These files support reproducibility, communication, and transparency in the project’s development process.
